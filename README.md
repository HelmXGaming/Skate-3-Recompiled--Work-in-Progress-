### Skate 3 Recompilation Project

This project provides a recompiled version of Skate 3 for Windows using UnleashedRecomp-style launcher logic. The launcher ensures required game assets are present, automatically extracts ISOs when needed, and provides clear error messages when files are missing.

### Quick Start Guide
1. **Clone the Repository**  
   ```bash
   git clone https://github.com/YourUser/Skate3Recomp.git
   cd Skate3Recomp
2. From the repo root, run the included script to configure the environment:
```
   setup.bat
```
3. Provide your Game Dump
	-Place your Skate 3 ISO in the GameAssets folder.
	-The launcher will automatically extract and verify the ISO contents on first run.
4. Open in Visual Studio
	- Launch Skate3Recomp.sln
	- Make sure you have the Desktop development with C++ workload installed.
	- Build the project in Release x64 mode.
5. Run the Launcher
	- After a successful build, run Launcher.exe.
	- If the required assets are missing, the launcher will exit with an error message.

### Requirements

Before building the project, ensure you have the following installed:

- **Visual Studio 2022 (17.4 or newer)**  
  - Workload: **Desktop development with C++**  
  - Components:  
    - MSVC v143 build tools (latest)  
    - Windows 10/11 SDK (10.0.22000 or newer recommended)  
    - C++ CMake tools for Windows (optional but useful)  

- **GitHub Desktop** (or Git CLI)  
  Used to clone and manage the repository.

- **Game Dump (ISO format)**  
  A clean dump of *Skate 3* (or other supported game).  
  - Must be placed in the `GameAssets` folder.  
  - The launcher will automatically extract assets using [extract-xiso](https://github.com/XboxDev/extract-xiso).  

- **Windows 10/11 (x64)**  
  Development and runtime environment.

### Setup

Clone this repository.

Run setup.bat in the project root.

Initializes Git

Configures CMake preset (Clang/LLVM + Ninja)

Downloads extract-xiso

Prepares the game_assets/ directory

Open the folder in Visual Studio.

Select the Skate3Launcher target and build.

### Adding the ISO

Place your Skate3.iso in the project root.

On first run, the launcher will extract the ISO into game_assets/ using extract-xiso.

If extraction fails or files are missing, the launcher will display an error and exit.

### Running

Build the project (Ctrl+Shift+B in Visual Studio).

Run Skate3Launcher.exe located in build/bin/.

If game_assets/ is empty, the ISO will be automatically extracted.

If no ISO or extracted files are found, the launcher exits with an error.

### Development Notes

Default toolchain: Clang/LLVM with Ninja

Alternative: MSVC x64 toolchain (slower, easier debugging)

Visual Studio Build Insights warnings can be safely ignored or disabled.

Launcher logic:

Check for game assets

Extract ISO if necessary

Launch recompiled Skate 3

Exit with clear error if missing files

### Project Goal

The goal of this project is to provide a standalone executable launcher for Skate 3. The user should be able to double-click the launcher, and it will:

Check for game_assets/.

Extract the ISO if necessary.

Launch the recompiled game.

Provide clear errors if assets are missing.
